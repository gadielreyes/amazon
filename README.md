# Amazon

## Description

Amazon Product Panel

### Prerequisites

- [docker](https://docs.docker.com/engine/install/ubuntu/)
- [docker-compose](https://docs.docker.com/compose/install/)

### Setpup Steps

1. `git clone git@github.com:gadielreyes/amazon.git`
2. `cd amazaon`
3. `make init`
4. `go to localhost:8000`
